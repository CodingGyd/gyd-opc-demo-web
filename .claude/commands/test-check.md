# /test-check — 测试自检流程

## 步骤

1. **文件完整性检查**：确保所有页面和组件的文件完整
   - Web：.vue / .ts
2. **配置检查**：确认 router 配置中的路径注册正确
3. **API 封装检查**：确认所有 API 文件的 baseURL 可配置、请求拦截器正常
4. **组件检查**：确认公共组件的 props / events 定义完整
5. **构建检查**：`npm run build` 确认构建成功

## 通过标准
- 所有页面和组件文件完整，无缺失
- 配置路径与实际文件一一对应
- API 封装统一使用 request 工具，自动携带 token
